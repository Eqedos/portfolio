# parthkumar.dev

Personal portfolio site built with Next.js 16, React 19, and Tailwind CSS.

## Features

- Terminal-themed UI with ASCII art name and interactive terminal emulator
- Mahoraga wheel of adaptation animation
- Work experience timeline with company logos
- Keyboard shortcut (`Ctrl+K`) to toggle terminal
- Terminal commands: `about`, `experience`, `projects`, `skills`, `links`, `contact`, `history`, `clear`

## Tech Stack

- **Framework**: Next.js 16 / React 19
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Language**: TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
